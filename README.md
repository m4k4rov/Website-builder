# Простой конструктор сайта о сериале (фильме и вообще о чём угодно) на JS.
Разметка страницы `index.html` содержит модуль <script> в котором выполняется функция `movieConstructor`
### Вот пример заполнения данной функции на примере сериала "Ведьмак" (* отмечены элементы, которые могут быть убраны):
  ```javascript
  movieConstructor('body', {                 // Вся разметка в данном примере добавляется в <body> 
      title: 'Ведьмак',                      // Название фильма (сериала). Служит заголовком страницы и еще некоторыми элементами
      background: 'witcher/background.jpg',  // Путь до изображения. Задний фон страницы.
      favicon: 'witcher/logo.png',           //* Путь до иконки. Иконка в заголовке страницы.
      fontColor: '#fff',                     //* Цвет текста
      backgroundColor: '#141218',            //* Цвет фона. В идеале должен совпадать с цветом края изображения заднего фона
      subColor: '#9D2929',                   //* Цвет интерактивных элементов (кнопок)
      header: {                                 //* Шапка сайта
        logo: 'witcher/logo.png',               //* Путь к логотипу сайта. Отображается в шапке сайта
        social: [                                  //* Модуль социальных сетей. Может быть любое количество блоков
          {
            title:'Twitter',                          // Наименование соц. сети
            link:'https://twitter.com',               // Ссылка на страницу соц
            image:'witcher/social/twitter.svg',       // Путь до иконки. Отображается в верхней строке сайта
          },
          {
            title:'Instagram',
            link:'https://instagram.com',
            image:'witcher/social/instagram.svg',
          },
          {
            title:'Facebook',
            link:'https://facebook.com',
            image:'witcher/social/facebook.svg',
          }
        ],
        menu: [                                                  //* Модуль с пунктами меню сайта. Может быть любое количество блоков   
          {                                                      
            title:'Описание',                                       // Текст пункта меню
            link:'https://www.kinopoisk.ru/series/1044004/',        // Ссылка для перехода
        },
          {
            title:'Трейлер',                                        // Для пункта меню "Трейлер" автоматически открывается модальное окно с видео по ссылке
            link:'https://www.youtube.com/watch?v=P0oJqfLzZzQ',
        },
          {
            title:'Отзывы',
            link:'https://www.kinopoisk.ru/film/1044004/reviews/',
        },
        ]
      },
      main: {                                                       // Центральная часть сайта
        genre: '2019, фэнтези',                                     // Дата выхода и жанр
        rating: '7',                                                // Рейтинг. Отображается в виде звёзд. От 1 до 10
        description: 'Ведьмак Геральт, мутант ... заколдованных принцесс.',   // Описание сериала
        trailer: 'https://www.youtube.com/watch?v=P0oJqfLzZzQ',               // Ссылка на трейлер сериала
        slider: [                                                   //* Слайдер с вышедшими сериями сериала. При клике на изображение открывается модальное окно
          {                                                         //Может быть любое количество блоков
            link: 'https://www.youtube.com/watch?v=P0oJqfLzZzQ',  //* При наличии ссылки на серию после клика на неё на сайте открывается модальное окно с просмотром
            img: 'witcher/series/series-1.jpg',                   // Путь до изображения. Отображается в слайдере
            title: 'Начало конца',                                //* Название серии 
            subtitle: 'Серия №1'                                  //* Номер серии
          },
          {
            img: 'witcher/series/series-2.jpg',
            title: 'Четыре марки',
            subtitle: 'Серия №2'
          },
          {
            img: 'witcher/series/series-3.jpg',
            title: 'Предательская луна',
            subtitle: 'Серия №3'
          },
          {
            img: 'witcher/series/series-4.jpg',
            title: 'Банкеты, ублюдки и похороны',
            subtitle: 'Серия №4'
          }
        ]
      },
      footer: {                                               //* Нижняя часть сайта
        copyright: '© 2020 The Witcher. All right reserved.', //* Торговый знак 
        footerMenu: [                                         //* Меню с ссылками  
          {
            title: 'Privacy Policy',                          // Текст пункта меню
            link: '#'                                         // Ссылка  
          },
          {
            title: 'Terms of Service',
            link: '#'
          },
          {
            title: 'Legal',
            link: '#'
          }
        ]
      }
    });
  ```
  
  ### Хотя большинство элементов может быть убрано, рекомендуется заполнять как можно больше полей для наполненности страницы.
